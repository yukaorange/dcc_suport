# After Effects: 法線マップ・深度マップを使った光エフェクト

ユーティリティマップ（法線マップ / 深度マップ / スペキュラマップ）をAE上で活用し、立体的なライティングやグロー演出を実現する手法集。

---

## 前提: ユーティリティマップとは

CG レンダリング時に出力できる補助マップ。コンポジット段階でライティングやエフェクトの制御に使う。

| マップ | エンコード内容 | 用途 |
|--------|--------------|------|
| Normal Map | 面の向き（R=X, G=Y, B=Z） | リライティング、凹凸表現 |
| Depth Map | カメラからの距離（白=近/黒=遠 or 逆） | 被写界深度、フォグ、奥行きグロー |
| Specular Map | 鏡面反射の強度 | ハイライト制御 |

---

## A. AE標準エフェクトのみ

### 1. 法線マップのチャンネル分離による疑似リライティング

法線マップの各チャンネルを分離して「光の当たる面」だけを選択的に照らす。

**手順:**

1. 法線マップレイヤーを配置
2. エフェクト > チャンネル > **チャンネルシフト（Shift Channels）** で特定チャンネルを抽出:
   - 左方向からの光 → Rチャンネル
   - 上方向からの光 → Gチャンネル
   - 正面からの光 → Bチャンネル
3. **レベル補正（Levels）** でコントラスト調整
4. 発光レイヤー（グロー、光色の平面など）の **ルミナンスマット** として使用
5. 光の方向アニメーション: R/G/Bマットの不透明度をエクスプレッションで時間変化させブレンド

### 2. 深度マップによる奥行きグロー

**方法A: フォグ（大気効果）**
- 白い平面のトラックマットに深度マップを設定し、不透明度で霧の濃さ調整

**方法B: 深度に応じたグロー差**
1. 元素材を複製、片方にエフェクト > スタイライズ > **グロー（Glow）** を適用
2. 深度マップをグローレイヤーのルミナンスマットに設定
3. 手前=弱グロー / 奥=強グロー（または逆）で奥行き感が出る

**方法C: Camera Lens Blur**
- エフェクト > ブラー&シャープ > **カメラレンズブラー**
- 「ブラーマップ」に深度マップを指定 → 奥行きに応じたボケで立体感強化

### 3. スペキュラマップの活用

- **Add（加算）モード** で元素材上に重ね → 光沢感
- レベル補正 + グロー で強度調整
- **CC Light Sweep** や **CC Light Burst** を適用 → 光の走査アニメーション

### 4. CC Glass / CC Plastic による疑似3Dライティング

法線マップに基づく凹凸ライティングの核心的手法。

**手順:**

1. 元素材にエフェクト > スタイライズ > **CC Glass** を適用
2. Bump Map: 法線マップレイヤーを指定
3. Light Direction: ライト方向を設定（キーフレームでアニメーション可能）
4. Light Intensity: 光の強度
5. Softness: 光の拡散度

**CC Plastic** はCC Glassの亜種で、よりプラスチック的な光沢。金属感のある指輪にはCC Glassの方が適している場合が多い。

### 5. ディスプレイスメントマップ（補助的）

- エフェクト > ディストーション > **ディスプレイスメントマップ**
- 深度マップを指定 → 平面素材に奥行き方向の歪み
- 光エフェクトと併用で立体感を強化

## B. サードパーティプラグイン

### 1. Normality（推奨 / 無料）

- 開発: Battle Axes
- 法線マップを直接読み込みリアルタイムリライティング
- 最大3灯のポイントライトを設定可能
- 光の方向・色・強度・フォールオフを制御
- AE上で法線マップリライティングを行う最もシンプルで強力な手段
- 対応: After Effects 2020+
- YouTube参考: 「After Effects Normality normal map lighting」で検索

### 2. Video Copilot Element 3D

- 3Dモデルを直接AE内で扱いライティング
- CG素材が既にレンダリング済みの場合はユーティリティマップ活用の方が効率的

### 3. Knoll Light Factory（Red Giant / Maxon）

- 高品質レンズフレア・ライトエフェクト生成
- 深度マップをマスク併用で奥行き感のあるフレア演出

### 4. RE:Map / RE:Lens（RE:Vision Effects）

- 法線マップを使ったリフレクション合成
- 高度なマップベースコンポジット向け

## C. 実践的ワークフロー例

CG素材 + depth/normal/specular の構成で「立体的な光の回り込み」を実現する手順:

1. **スペキュラ層**: スペキュラマップを Add モードで重ね、CC Light Sweep で光走査アニメーション
2. **法線ライティング層**: CC Glass の Bump Map に法線マップを指定、Light Direction をアニメーション（「光の回り込み」の核）
3. **深度グロー層**: 深度マップをルミナンスマットにしてグローの強弱を制御
4. **被写界深度**: Camera Lens Blur で深度マップを使いボケを追加
5. **仕上げ**: カラー補正・全体のグロー微調整

プラグイン導入可能なら **Normality** で手順2を置き換えると、より直感的で高品質なリライティングが可能。
