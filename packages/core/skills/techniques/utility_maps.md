# ユーティリティマップ（Utility Maps）

CGレンダリング時に出力する補助的なパス。コンポジット段階でライティング・エフェクト制御に利用する。

## 主要なユーティリティマップ

### Normal Map（法線マップ）
- 各ピクセルの面の向きをRGBカラーでエンコード
- R = X方向（左右）、G = Y方向（上下）、B = Z方向（前後/カメラ方向）
- 用途: リライティング（光の方向変更）、凹凸表現の制御
- 特徴: チャンネルを分離することで「特定方向を向いた面」だけを選択できる

### Depth Map（深度マップ）
- カメラからの距離をグレースケールで表現
- 白=近/黒=遠（またはその逆、レンダラー設定による）
- 用途: 被写界深度（DoF）、フォグ/大気効果、奥行きに応じたエフェクト強弱

### Specular Map（スペキュラマップ）
- 鏡面反射の強度をエンコード
- 用途: ハイライト・光沢の制御、光の走査アニメーションのベース

### その他
- **AO（Ambient Occlusion）**: 環境遮蔽。接触面の暗さを表現
- **ID Map / Object ID**: オブジェクトごとの色分け。マスク生成に使用
- **Motion Vector**: ピクセルの移動方向・速度。モーションブラーに使用
- **Position Pass**: ワールド座標をRGBでエンコード。3D空間でのマスキングに使用

## コンポジットでの基本的な使い方

1. **マットとして使う**: ルミナンスマットやアルファマットとして設定し、他レイヤーの表示範囲を制御
2. **エフェクトの入力マップとして使う**: Camera Lens BlurのBlur Map、Displacement MapのMap Layerなど
3. **加算合成で重ねる**: スペキュラマップなどをAddモードで重ねてハイライト追加
4. **プラグインの入力として使う**: Normalityなど法線マップ対応プラグインに直接渡す
