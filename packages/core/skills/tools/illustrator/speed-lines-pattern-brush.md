# Illustrator: 集中線（スピードライン）— パターンブラシ方式

## 概要
直線に線幅プロファイルを適用し、ランダム変形後にパターンブラシとして登録。正円に適用することで集中線を一発生成する手法。後からサイズ・色の変更が容易。

## 手順

### 1. 元となる直線を作成
- **直線ツール(¥ / Line Segment Tool)** で垂直線を描画（長さ700px程度、線幅7pt程度）
- **線パネル(Stroke Panel)** > **線幅プロファイル(Width Profile)** から「線幅プロファイル4(Width Profile 4)」を選択（先端が細くなる形状）

### 2. 複数コピーして並べる
- **選択ツール(V)** で直線を選択 → **Return(Enter)** で「移動(Move)」ダイアログを表示
- 水平方向: 5px、垂直方向: 0px → **コピー(Copy)** をクリック
- **Cmd+D (Ctrl+D)** を約10回繰り返して等間隔に複製

### 3. 透明な境界線を追加（ブラシの余白制御用）
- 一番右端の線をAlt(Option)+ドラッグで一番左端に移動コピー
- コピーした線の色を「なし」に設定 → パターンブラシの端に余白ができる

### 4. 個別に変形でランダム感を出す
- 全ての線を選択
- **オブジェクト(Object)** > **変形(Transform)** > **個別に変形(Transform Each)**
  - 拡大・縮小 水平方向: 10%、垂直方向: 60〜80%
  - 移動 水平方向: 20px、垂直方向: 0px
  - **「ランダム(Random)」にチェック**
  - **「線幅と効果を拡大・縮小(Scale Strokes & Effects)」にチェック**
  - 基準点: 上中央
- OK

### 5. パターンブラシとして登録
- 全選択のまま **ブラシパネル(Brushes Panel)** > **新規ブラシ(New Brush)** > **パターンブラシ(Pattern Brush)**
- 間隔(Spacing): 10%程度（プレビューで調整）
- 着色の方式(Colorization Method): **色相のシフト(Hue Shift)** → 後から色変更可能に

### 6. 正円に適用して集中線を生成
- **楕円形ツール(L)** + Shift で正円を描画
- ブラシパネルから登録したパターンブラシをクリックして適用
- 必要に応じてクリッピングマスクで外周をトリミング

## 表現のコツ
- 線の太さ・長さのバラつきが大きいほど「漫画的な迫力」が出る
- リファレンスのような高密度な集中線にするには、元の線の本数を増やすか、間隔を狭める
- 着色方式を「色相のシフト」にしておくと、線の色を黒→赤などに簡単に変更できる
