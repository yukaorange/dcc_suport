# Photoshop 主要ショートカット

## ツール
| キー | ツール |
|------|--------|
| V | 移動ツール（Move） |
| M | 選択ツール（長方形/楕円） |
| L | なげなわツール |
| W | クイック選択/自動選択 |
| C | 切り抜きツール（Crop） |
| I | スポイトツール（Eyedropper） |
| B | ブラシツール |
| S | スタンプツール（Clone Stamp） |
| E | 消しゴムツール |
| G | グラデーション/塗りつぶし |
| P | ペンツール |
| T | 文字ツール |
| U | シェイプツール |

## 頻用操作
| ショートカット | 操作 |
|----------------|------|
| Ctrl/Cmd+Z | 取り消し（複数回可） |
| Ctrl/Cmd+T | 自由変形 |
| Ctrl/Cmd+J | レイヤーを複製 |
| Ctrl/Cmd+G | レイヤーをグループ化 |
| Ctrl/Cmd+E | レイヤーを結合 |
| Ctrl/Cmd+Alt/Opt+G | クリッピングマスク作成/解除 |
| Ctrl/Cmd+L | レベル補正 |
| Ctrl/Cmd+M | カーブ |
| Ctrl/Cmd+U | 色相・彩度 |
| [ / ] | ブラシサイズ縮小/拡大 |
| D | 描画色/背景色をデフォルトに |
| X | 描画色と背景色を入れ替え |

## 表示
| ショートカット | 操作 |
|----------------|------|
| Ctrl/Cmd+0 | 全体を表示 |
| Ctrl/Cmd+1 | 100%表示 |
| Tab | パネルの表示/非表示 |
| F7 | レイヤーパネル |
| Space+ドラッグ | 手のひらツール一時切替 |
