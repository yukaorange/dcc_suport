# Photoshop レイヤー操作

## レイヤーパネルの基本
- ウィンドウ > レイヤー（F7）で表示
- 目のアイコン: 表示/非表示 / 鍵アイコン: ロック
- 不透明度: 上部スライダー / 描画モード: 不透明度左のドロップダウン

## スマートオブジェクト
- レイヤー右クリック > スマートオブジェクトに変換
- 拡大縮小で劣化しない。スマートフィルターで非破壊フィルター適用可能
- ダブルクリックで元データを別ウィンドウで編集

## 調整レイヤー
- レイヤーパネル下部の「半円アイコン」またはレイヤー > 新規調整レイヤー
- 非破壊編集。後から設定変更可能
- クリッピングマスクで直下のレイヤーのみに適用: Ctrl/Cmd+Alt/Opt+G

## レイヤーマスク
- レイヤーパネル下部の「矩形に丸のアイコン」
- 白: 表示、黒: 非表示、グレー: 半透明
- Shift+クリック: マスクの有効/無効切替
- Alt/Opt+クリック: マスクのみを表示

## クリッピングマスク
- 下のレイヤーの形状に上のレイヤーを合わせる
- Ctrl/Cmd+Alt/Opt+G またはレイヤー境界線をAlt/Opt+クリック

## レイヤースタイル
- レイヤーをダブルクリック
- ドロップシャドウ / 光彩（外側/内側） / ベベルとエンボス
- グラデーションオーバーレイ / パターンオーバーレイ / 境界線
- 右クリック > スタイルをコピー/ペーストで他レイヤーに転用

## グループ
- Ctrl/Cmd+G でグループ化
- グループにもマスク・ブレンドモード・不透明度が設定可能
