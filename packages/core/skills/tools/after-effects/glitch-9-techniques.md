# グリッチ表現 9種のノイズ素材

## 概要
After Effects の標準エフェクトのみで作るグリッチ表現の9素材集。スキャンライン、波形ワープ、データ破損、色収差、ゴースト、RGBブロックノイズ等を組み合わせて、レトロCRT〜デジタルバグ表現まで幅広く構築する。

出典: https://www.youtube.com/watch?v=zyRCDYmO1VQ

## 共通の準備
- コンポジション `GLITCH_render`: 1920x1080 / 30fps / 10秒 / 背景黒
- 元素材を `GLITCH_Footage` としてプリコンポーズ

## ① Base Scan Line
CRT風の静止水平走査線。

- 調整レイヤー `1. Base Scan Line`
- Effect > 描画 > **グリッド**
  - サイズ: 幅と高さスライダー
  - 幅: `40000`（垂直線を画面外へ）
  - 高さ: `3`、ボーダー: `4`
  - ぼかし高さ: `2`
  - 描画モード: **ステンシルアルファ**

## ② Scan Line Up / Down
上下に流れる点滅走査線。

- 調整レイヤー `2. Scan Line Up`
- Effect > ノイズ&グレイン > **フラクタルノイズ**
  - 種類: ブロック / 複雑度: 1
  - 縦横比固定オフ → スケール幅 `100000` / 高さ `20`
  - 不透明度キーフレーム: 0f=30% / 3f=15% / 6f=30%（**ホールド**）
    - エクスプレッション: `loopOut(type="cycle", numKeyframes=0)`
  - 乱気流オフセットY:
    ```
    x=effect("フラクタルノイズ")("乱気流のオフセット")[0];
    y=effect("フラクタルノイズ")("乱気流のオフセット")[1];
    y-=time*60; [x,y]
    ```
  - 描画モード: オーバーレイ
- `2. Scan Line Down` は複製してスケール高さ `150`、オフセットを `y+=time*30` に

## ③ Waveform Warp
水平方向に波打つ歪み。

- 調整レイヤー `3. Waveform Warp`
- Effect > ディストーション > **波形ワープ**
  - 種類: ノイズ / 高さ: 12 / 幅: 50 / 速度: 0
  - 高さにエクスプレッション: `noise(time*15)*40`
- Effect > スタイライズ > **モーションタイル**
  - 出力幅/高さ: 150 / ミラーエッジ: ON

## ④ Deform Line
ノイズマップで水平方向データをずらす。

- 平面 `noise_tex`（50%グレー）
- フラクタルノイズ
  - 種類: ブロック / コントラスト: 70 / 複雑度: 1
  - スケール幅 `5000` / 高さ `600`
  - 乱気流オフセットY: `time*-100`
- → プリコンポーズ `RGB_Block_Noise_tex`
- 調整レイヤー `4. Deform Line`
- Effect > ディストーション > **ディスプレイスメントマップ**
  - レイヤー: `RGB_Block_Noise_tex`
  - 水平/垂直とも輝度参照
  - 最大水平: 80 / 最大垂直: 14
- モーションタイル: 150/150/ミラー
- 派生: `最大水平置換` に `noise(time*5)*100` でアニメ化

## ⑤ Grain Noise
フィルムグレイン質感。

- 平面 `5. Grain Noise`（50%グレー）
- Effect > ノイズ&グレイン > **ノイズ**
  - 量: 100% / カラーノイズ: ON
- 描画モード: オーバーレイ
- ブラー(ガウス) 5
- ズーム: 拡大率 160 / サイズ 4000

## ⑥ Tri Tone
3トーン着色。

- 調整レイヤー `6. Tri Tone`
- Effect > カラー補正 > **トライトーン**
  - 例（青系）: ハイライト `#99E7FE` / ミッド `#0067FF` / シャドウ `#00018C`

## ⑦ Chromatic Aberration
RGBチャンネルずらしによる色収差。

- ①〜⑥を `GLITCH_base` としてプリコンポーズ
- `GLITCH_base` を3枚重ねる
- 各レイヤーに **チャンネルシフト** で R/G/B 以外をフルオフ
- 各レイヤーに **モーションタイル** 150/150/ミラー
- タイルの中心エクスプレッション（Rの例）:
  ```
  posterizeTime(10);
  x=value[0]+noise(time*5)*20;
  y=value[1]+noise(time*6)*20;
  [x,y]
  ```
  G/B はシード値（noiseの倍率）と振幅を変える
- 上2枚を**加算**ブレンド
- メインコンポでは描画モード **ピンライト**

## ⑧ Ghost
ランダム位置にワープする残像。

- `GLITCH_base` を複製 → `8. Ghost`（最上段）
- モーションタイル 150/150/ミラー
- タイル中心: `posterizeTime(3); [random(0,1080), random(0,1080)]`
- 不透明度: `posterizeTime(10); Math.floor(random(2))*100`
- 描画モード: オーバーレイ

## ⑨ RGB Block Noise
ブロック状のバグノイズ。

- 新規コンポ `9. RGB_Block_Noise` に `GLITCH_Footage` を3枚配置
- 各レイヤーで **チャンネルシフト**（R/G/B分離）+ **ディスプレイスメントマップ**（map: `RGB_Block_Noise_tex`）
  - R: 水平 +20 / 垂直 0
  - G: 水平 -20 / 垂直 0
  - B: 水平 0 / 垂直 +20
- 各レイヤーにモーションタイル 150/150/ミラー
- 描画モード: 加算
- メインコンポで描画モード: オーバーレイ
- 仕上げ: 色相/彩度（彩度+40）、アンシャープマスク 200
- 顔保護: `RGB_Block_Noise_tex` 内に黒平面 + 楕円マスク反転 + ぼかし
- 不透明度: `posterizeTime(10); Math.floor(random(2))*100`

## 設計上のポイント

- **走査線・グレイン** はベース質感、**波形ワープ・Deform Line・RGBブロック** は破壊的演出、**色収差・ゴースト** は仕上げの奥行き、という三層構造
- `posterizeTime(n)` を噛ませると「カクつき＝壊れた感」が出る。グリッチ系エクスプレッションの定番
- ノイズマップ（`RGB_Block_Noise_tex`）を **ディスプレイスメントマップ** で使い回すことで、Deform LineとRGB Block Noiseが同じパターンに同期し統一感が出る
- 描画モード設計: 走査線=ステンシルアルファ/オーバーレイ、色収差レイヤー=加算→ピンライト、ゴースト=オーバーレイ
