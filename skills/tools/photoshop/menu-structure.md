# Photoshop メニュー構造

## ファイル（File）
- 新規（New）: Ctrl/Cmd+N
- 開く（Open）: Ctrl/Cmd+O
- 別名で保存（Save As）: Ctrl/Cmd+Shift+S
- 書き出し > 書き出し形式（Export > Export As）
- 書き出し > クイック書き出し（Quick Export as PNG）

## 編集（Edit）
- 取り消し（Undo）: Ctrl/Cmd+Z
- 自由変形（Free Transform）: Ctrl/Cmd+T
- 変形（Transform）> 拡大縮小/回転/ゆがみ/遠近法
- パターンを定義（Define Pattern）

## イメージ（Image）
- モード（Mode）: RGB/CMYK/グレースケール切替
- 色調補正（Adjustments）> レベル補正/カーブ/色相・彩度 等（直接適用=破壊的）
- 画像解像度（Image Size）: Alt/Opt+Ctrl/Cmd+I
- カンバスサイズ（Canvas Size）: Alt/Opt+Ctrl/Cmd+C

## レイヤー（Layer）
- 新規レイヤー / レイヤーを複製 / レイヤーを結合（Ctrl/Cmd+E）
- スマートオブジェクトに変換（Convert to Smart Object）
- レイヤーマスク（Layer Mask）> 表示・非表示
- クリッピングマスクを作成: Ctrl/Cmd+Alt/Opt+G
- レイヤースタイル > ドロップシャドウ/ベベル/グラデーションオーバーレイ等
- 新規調整レイヤー（New Adjustment Layer）

## フィルター（Filter）
- ぼかし（Blur）> ぼかし（ガウス）/ 動き / レンズ
- シャープ（Sharpen）> アンシャープマスク
- ノイズ（Noise）> ノイズを加える / ノイズを軽減
- フィルターギャラリー / Camera Raw フィルター / ゆがみ / ニューラルフィルター

## 選択範囲（Select）
- すべてを選択: Ctrl/Cmd+A / 選択を解除: Ctrl/Cmd+D
- 選択範囲を反転: Ctrl/Cmd+Shift+I / 選択とマスク

## ウィンドウ（Window）
- レイヤー: F7 / プロパティ / ヒストリー / カラー: F6 / ブラシ: F5
