# Photoshop 星空・宇宙背景テクニック

Source (動画要約済み): [How to create a synthetic/cosmic starfield in photoshop](https://www.youtube.com/watch?v=kU-dUE-I8CE)

## 概要
ノイズ、ぼかし、レベル補正、雲フィルター、複数の描画モードを組み合わせて
奥行きのある幻想的な星空・星雲背景を作成する手法。プラグイン不要。

## 手順（動画準拠）

### Step 1: 新規ドキュメント
- File > New (Ctrl+N) → 1920×1080px, 72ppi, RGB 8bit, 背景: 黒

### Step 2: 星空ベース
- Backgroundレイヤーを右クリック → **スマートオブジェクトに変換（Convert to Smart Object）**
- **フィルター > ノイズ > ノイズを加える（Filter > Noise > Add Noise）**
  - 量: 200%（画像サイズが大きいほど値を上げる）
  - 分布: 均等（Uniform）
  - グレースケールノイズ: ON
- **フィルター > ぼかし > ぼかし（ガウス）（Filter > Blur > Gaussian Blur）**
  - 半径: 2.0px

### Step 3: 星のコントラスト・色調整
- **レベル補正（Levels）** 調整レイヤーを追加
  - シャドウ入力: 6 / 中間調: 1.00 / ハイライト入力: 255
- Levelsレイヤーを右クリック → **描画オプション（Blending Options）**
  - **グラデーションオーバーレイ（Gradient Overlay）** ON
    - 描画モード: 覆い焼き（リニア）- 加算 / Linear Dodge (Add)
    - 不透明度: 100%、スタイル: 線形、角度: 45°
    - グラデーション: 青系プリセット
- Levelsレイヤーを **クリッピングマスク** に設定（Layer 1にクリップ）

### Step 4: 光源（レンズフレア）
- **フィルター > 描画 > 逆光（Filter > Render > Lens Flare）**
  - 明るさ: 100%、レンズ: 35mm Prime、位置: 右中央付近

### Step 5: 色相・彩度の全体調整
- **色相・彩度（Hue/Saturation）** 調整レイヤー
  - 色彩の統一（Colorize）: ON
  - 色相: 215 / 彩度: 45 / 明度: 0

### Step 6: 星雲効果 (1) — 暗部の深み
- 新規レイヤー「Cloud」
- 前景色: #001540 / 背景色: #ffffff
- **フィルター > 描画 > 雲模様1（Filter > Render > Clouds）**
- 描画モード: **乗算（Multiply）**

### Step 7: 星雲効果 (2) — 色のグラデ
- 新規レイヤー「Color」
- 前景色: #ff0000 / 背景色: #7200ff
- グラデーションツール（G）で左下→右上にドラッグ
- **ぼかし（ガウス）150px**
- 描画モード: **オーバーレイ（Overlay）**

### Step 8: 星雲効果 (3) — 発光
- 新規レイヤー「Burn Cloud」
- 前景色: #001540 / 背景色: #ffffff
- **雲模様1（Clouds）**
- 描画モード: **覆い焼きカラー（Color Dodge）**

## 描画モード早見表

| テクニック | 描画モード | 効果 |
|-----------|-----------|------|
| ノイズ + ぼかし + レベル | - | 星の自然なランダム分布 |
| グラデーションオーバーレイ | Linear Dodge (Add) | 星の色彩付け（暗部透過） |
| 雲模様1 | Multiply | 暗い領域の深み・星の密度ムラ |
| グラデーション | Overlay | 全体の色合い混合 |
| 雲模様1 | Color Dodge | 明部の発光・星雲の輝き |

## 今回プランへの応用メモ
- この動画はベースが**黒背景**。今回のリファレンスは**黄昏グラデーション**が土台
- ステップ1でグラデーション背景を先に作り、ステップ5でこの星テクニックを**スクリーン合成**で重ねる
- Step 6-8の星雲テクニック（雲模様+描画モード）はステップ2のフォグ制作にそのまま応用可能
- 色味をリファレンス寄り（モーヴ・パープル系）に置き換えて使う
- グロウのLinear Dodgeはグラデ背景の明るい領域で飛びやすいので、不透明度を下げて調整
