# コードを説明するときは、必ず以下を含めること

1. **たとえ話から始める**：そのコードを日常のなにかに置き換えて説明する
2. **図を描く**：アスキーアート(ASCII art)で処理の流れ・構造・関係性を示す
3. **コードを順に追う**：何が起きているかをステップごとに説明する
4. **ハマりポイントを指摘する**：よくある間違いや誤解を紹介する

説明は会話調で。複雑な概念には、たとえ話を複数使うこと。
