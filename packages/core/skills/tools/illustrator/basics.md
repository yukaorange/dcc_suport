# Adobe Illustrator 基本操作

## オブジェクトの重ね順（Zオーダー）
- `オブジェクト(O)` > `重ね順(R)` > `最前面へ` (Shift+Cmd+]) / `最背面へ` (Shift+Cmd+[)
- 一段ずつ移動: `前面へ` (Cmd+]) / `背面へ` (Cmd+[)

## 塗りと線
- カラーパネル: `ウィンドウ(W)` > `カラー` (F6)
- パネル内の「塗り」アイコン（塗りつぶし四角）または「線」アイコン（枠線四角）をクリックしてアクティブにしてから色を変更
- 「なし」は斜線入りの四角アイコン

## レイヤー操作
- レイヤーパネル: `ウィンドウ(W)` > `レイヤー` (F7)
- 新規レイヤー: パネル下部の「＋」ボタン
- 表示/非表示: レイヤー名左の目玉マーク
- ロック: 目玉マーク右隣の空欄をクリック → 鍵マークが表示され編集不可になる
- 重なり順変更: レイヤーをドラッグ＆ドロップ

## 文字のワープ変形
- テキストを選択 → `オブジェクト(O)` > `エンベロープ(E)` > `ワープで作成(M)`
- `ワープオプション` で「スタイル: 弧」を選択し、カーブスライダーで調整
- 別の方法（効果ベース）: `効果(Effect)` > `ワープ(Warp)` > `アーチ(Arc)` → カーブ値で湾曲度調整

## 不透明度の変更
- オブジェクトを選択 → プロパティパネルまたは透明パネルの「不透明度」スライダーを調整
- `ウィンドウ(W)` > `透明(Transparency)` でパネル表示

## ショートカット
| 操作 | Mac | Win |
|------|-----|-----|
| カラーパネル表示 | F6 | F6 |
| レイヤーパネル表示 | F7 | F7 |
| 最前面へ | Shift+Cmd+] | Shift+Ctrl+] |
| 最背面へ | Shift+Cmd+[ | Shift+Ctrl+[ |
| 前面へ | Cmd+] | Ctrl+] |
| 背面へ | Cmd+[ | Ctrl+[ |

## ポイント
- Illustratorはベクターツール。拡大縮小しても劣化しないためロゴ・印刷物に最適
- すべての要素は「オブジェクト」として独立しており、積層構造でデザインが構成される
- 複雑なファイルは背景レイヤーをロックして誤操作を防ぐ習慣をつける
- Illustratorが得意: ロゴ・アイコン・文字アレンジ・図形 / 苦手: 写真加工（→Photoshop）
